## 📱 About

**Axion K** is an Android AI assistant powered by Groq API. 
Clean architecture, custom gold-accented theme, smooth animations, 
and a polished UX — all written by a 15-year-old developer.

Author: **Evgeniy Kramer**, 15 y.o.

## ✨ Features

- 💬 Chat with Llama 3 / Mixtral / Gemma via Groq API
- 🎨 Custom theme engine — users pick their own colors
- 🧠 System prompts & AI personality customization
- 📎 Markdown rendering (bold, code blocks, lists)
- 📱 Jetpack Compose with typewriter animation
- 🔒 Local auth with SHA-256 password hashing
- 📂 Chat history with search
- 🎤 Voice input
- 📸 Image recognition

## 🛠 Tech Stack

| Technology | Purpose |
